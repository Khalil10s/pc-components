Dator Komponenter

Utforska de viktigaste komponenterna som utgör en modern dator. Klicka på en komponent för att avslöja mer detaljer, eller använd sökfältet för att hitta specifika delar.

CPU

Central Processing Unit, datorns hjärna. Hanterar alla instruktioner från programvara.

  • Märken: Intel, AMD
  • Nyckelspecifikationer: Kärnor, Trådar, Klockhastighet, Cache
  • Användning: Viktig för alla databehandlingar

GPU

Graphics Processing Unit, hanterar grafikrendering och parallell bearbetning.

  • Märken: NVIDIA, AMD
  • Specifikationer: VRAM, CUDA Kärnor, Klockhastighet
  • Användning: Spel, videoredigering, AI

RAM

Random Access Memory, tillfällig minne för att köra program snabbt.

  • Typer: DDR4, DDR5
  • Specifikationer: Kapacitet (8GB-64GB), Hastighet (MHz)
  • Användning: Multitasking, prestanda

Case

Skyddar komponenter och ger luftflöde för kylning.

  • Storlekar: ATX, Micro-ATX, Mini-ITX
  • Funktioner: RGB-belysning, härdat glas
  • Användning: Estetik och kylning

Motherboard

Ansluter alla komponenter och möjliggör kommunikation mellan dem.

  • Märken: ASUS, MSI, Gigabyte
  • Formfaktorer: ATX, Micro-ATX
  • Användning: Kompatibilitetshubb

Storage

Lagrar data permanent (SSD för hastighet, HDD för kapacitet).

  • Typer: SSD, HDD
  • Storlekar: 256GB-2TB SSD, 1TB-4TB HDD
  • Användning: Datalagring och uppstartshastighet

PSU

Power Supply Unit, levererar ström till alla datorns komponenter.

  • Effektivitet: 80 Plus-certifiering
  • Effekt: 500W-1000W
  • Användning: Stabil strömförsörjning
