Computer Components

Explore the essential components that make up a modern computer. Click on any component to reveal more details, or use the search bar to find specific parts.

CPU

Central Processing Unit, brain of the computer. Handles all instructions from software.

  • Brands: Intel, AMD
  • Key Specs: Cores, Threads, Clock Speed, Cache
  • Use: Essential for all computing tasks

GPU

Graphics Processing Unit, handles graphics rendering and parallel processing.

  • Brands: NVIDIA, AMD
  • Specs: VRAM, CUDA Cores, Clock Speed
  • Use: Gaming, video editing, AI

RAM

Random Access Memory, temporary memory for running programs quickly.

  • Types: DDR4, DDR5
  • Specs: Capacity (8GB-64GB), Speed (MHz)
  • Use: Multitasking, performance

Case

Protects components and provides airflow for cooling.

  • Sizes: ATX, Micro-ATX, Mini-ITX
  • Features: RGB lighting, tempered glass
  • Use: Aesthetics and cooling

Motherboard

Connects all components together and allows communication between them.

  • Brands: ASUS, MSI, Gigabyte
  • Form Factors: ATX, Micro-ATX
  • Use: Compatibility hub

Storage

Stores data permanently (SSD for speed, HDD for capacity).

  • Types: SSD, HDD
  • Sizes: 256GB-2TB SSD, 1TB-4TB HDD
  • Use: Data storage and boot speed

PSU

Power Supply Unit, supplies power to all computer components.

  • Efficiency: 80 Plus certification
  • Wattage: 500W-1000W
  • Use: Stable power delivery
